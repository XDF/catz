
===== 2012-02-26

Here are the basic commands to install the Perl modules used by catz.

All enviroments (dev and prod) are now using Perl 5.14.2.

In case of trouble, use
 perl -MCPAN -e "CPAN::Shell->force(qw(install Module::In::Question));"
that was found here: http://www.perlmonks.org/?node_id=606768 

cpan
install CPAN
reload cpan
o conf auto_commit 1
o conf prerequisites_policy follow
o conf build_requires_install_policy yes
install YAML
install LWP
install Bit::Vector
install Crypt::Blowfish
install Crypt::CBC
install DateTime
install Digest::HMAC
install Digest::MD5
install DBI
install DBD::SQLite
install Email::Date::Format
install File::Slurp
install GD
install HTTP::Date
install I18N::AcceptLanguage
install IO::Epoll
install EV
install Mojolicious
install Number::Format
install URI:Escape::XS
install XML::RSS
install Cache::Memcached::Fast
install Image::Size
install Image::ExifTool
install List::MoreUtils
install Text::LevenshteinXS
install Const::Fast
install Perl::Tidy
install Perl::Critic
install WWW::Mechanize
install HTML::Lint
install DateTime::Format::W3CDTF
install MIME::Base64
quit
