
===== 2013-03-27

Here are the basic commands to install the Perl modules used by catz.

All environments use Perl 5.16.2.

In case of trouble, use
 perl -MCPAN -e "CPAN::Shell->force(qw(install Module::In::Question));"
that was found here: http://www.perlmonks.org/?node_id=606768 

cpan
install CPAN
install CPAN:SQLite
install YAML
install LWP
reload cpan
o conf auto_commit 1
o conf prerequisites_policy follow
o conf build_requires_install_policy yes
exit

cpan
