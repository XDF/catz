
===== 2014-07-20

Here are the basic commands to install the Perl modules used by catz.

Development enviroment Perl 5.18.2
Production enviroment Perl 5.20.0

In case of trouble, use
 perl -MCPAN -e "CPAN::Shell->force(qw(install Module::In::Question));"
that was found here: http://www.perlmonks.org/?node_id=606768 

  cpan
  o conf auto_commit 1
  o conf prerequisites_policy follow
  o conf build_requires_install_policy yes
  install CPAN
  install CPAN::SQLite
  install YAML
  install LWP
  reload cpan
  exit

Then install all the required modules.

    perl -MCPAN -e"install Bit::Vector, Cache::Memcached::Fast, Crypt::Blowfish, Crypt::CBC, DateTime, Digest::HMAC, Digest::MD5, DBI, DBD::SQLite, Email::Date::Format, File::Slurp, GD, HTTP::Date, I18N::AcceptLanguage, IO::Epoll, EV, Mojolicious, Number::Format, URI::Escape::XS, XML::RSS, Image::Size, Image::ExifTool, List::MoreUtils, Text::LevenshteinXS, Const::Fast, Perl::Tidy, Perl::Critic, WWW::Mechanize, HTML::Lint, DateTime::Format::W3CDTF, MIME::Base64"

On Windows: IO:Epoll won't install (Linux only). Install Win32::Console to get rid of warning when running tests.

   perl -MCPAN -e"install Win32::Console"

2014-03-05: These were necessary on Windows:

   perl -MCPAN -e "CPAN::Shell->force(qw(install GD));"
   perl -MCPAN -e "CPAN::Shell->force(qw(install Number::Format));"
