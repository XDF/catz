2015-04-12

This folder contains the source data files (structured tagged text) that are 
the primary data source of https://catza.net/ service. The data from the files
gets offline loaded to the database by a loader program and is served from 
database at runtime. The news articles file metanews.txt is deliberately 
excluded.

All data and data files are licensed under the CC BY 4.0
https://creativecommons.org/licenses/by/4.0/ 

Notice that the files here change less often than they are updated in our data
process. The files here are updated only when a new version of Catz web
application engine is committed.

2021-04-25: These files are the final version. Data is no longer updated.
